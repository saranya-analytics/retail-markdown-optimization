🛒 Retail Markdown & Price Optimization Analytics
An end-to-end data analytics project focused on evaluating the impact of promotional markdowns on retail sales performance, optimizing discount strategies, and presenting actionable business insights through an interactive Power BI dashboard.
📌 Project Overview
Retailers frequently offer markdowns to boost sales volume, but untargeted discounts can lead to profit margin erosion. This project analyzes promotional data to identify which markdown strategies drive high ROI, customer engagement, and overall revenue growth without sacrificing profitability.
🛠️ Tech Stack & Tools
* **Data Processing & Cleaning:** Python (`pandas`, `numpy`) via Google Colab / Cloud Environment
* **Data Visualization & Dashboarding:** Microsoft Power BI
* **Documentation & Version Control:** Git & GitHub
📂 Repository Structure
data_cleaning.ipynb # Python data cleaning & preprocessing notebook (Google Cloud)
cleaned_markdown_dataset.csv # Cleaned dataset extracted from Google Cloud
retail-optimization-dashboard.pbix # Interactive Power BI Dashboard file
dashboard.png # Screenshot preview of the Power BI dashboard
README.md # Project documentation
📊 Key Business Insights & Findings
* **Markdown Efficiency:** Certain promotional tiers generated a 25%+ surge in sales volume while maintaining acceptable profit margins.
* **Seasonal Trends:** Peak promotional responsiveness was observed during specific holiday windows and end-of-quarter clearouts.
* **Category Performance:** High-margin product categories required lower discount percentages to achieve target sales velocity compared to low-margin items.
